All about HTML

HTML stands for Hyper Text Markup Language, and its responsible for creating the "items" you see on a webpage. For example, the 'World' button and the text you see here is created using the '<button>' '<p>' tag. Each html tag can also come with attribute that adjust the 'settings' of that item. For example, the 'src' attribute of the '<img>' tag controls the source of the image, and the 'width'&'height' attribute controls the size of the image.

All about CSS

While HTML controls whats items are on the page, CSS on the other hand controls how each item will look like ( or the style they have).
